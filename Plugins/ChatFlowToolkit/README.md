# ChatFlowToolkit

ChatFlowToolkit 是一个处理对话流输入的工具箱，可以把用户的输入文字和上传的文件分开。

## 项目结构

- `metadata.yaml`: 插件元数据
- `introduction.md`: 插件在商店页面的介绍
- `ParseUserInput`:
  - `main.ts`: 工具源代码
  - `metadata.yaml`: 工具元数据
  - `test-USER_INPUT.txt`: 用于测试的 `USER_INPUT` 参数
- `ExtractFiles`:
  - `main.ts`: 工具源代码
  - `metadata.yaml`: 工具元数据
